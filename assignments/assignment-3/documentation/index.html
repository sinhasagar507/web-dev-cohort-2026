Introduction to Mintlify

Build beautiful documentation that converts users

What is Mintlify?

Mintlify is the intelligent documentation platform that helps teams create and maintain world-class documentation built for both humans and AI. Our platform integrates seamlessly with your existing workflow and provides powerful tools to make your documentation shine.

Key Features

  • AI-Powered Writing: Get intelligent suggestions and automated content generation
  • Beautiful Themes: Choose from professionally designed themes or customize your own
  • Developer-Friendly: Write in Markdown with powerful extensions
  • Fast Performance: Lightning-fast sites with excellent SEO

Getting Started

Follow these steps to create your first documentation site with Mintlify:

1

Install Mintlify CLI

npm install -g mintlify
2

Initialize Your Project

mintlify init
3

Start Development Server

mintlify dev
4

Deploy Your Site

mintlify deploy

Configuration

Customize your documentation site by editing the mint.json configuration file:

{
  "name": "My Documentation",
  "logo": {
    "light": "/logo/light.png",
    "dark": "/logo/dark.png"
  },
  "favicon": "/favicon.png",
  "colors": {
    "primary": "#1a1a1a",
    "light": "#f8f9fa",
    "dark": "#1a1a1a"
  },
  "navigation": [
    {
      "group": "Getting Started",
      "pages": [
        "introduction",
        "quickstart",
        "installation"
      ]
    }
  ]
}
